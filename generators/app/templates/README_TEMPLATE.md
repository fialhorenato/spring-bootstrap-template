This generator scaffolds the Spring Bootstrap Kotlin project.

Placeholders replaced:
- __PROJECT_NAME__
- __GROUP_ID__
- __ARTIFACT_ID__
- __PACKAGE_NAME__
- __DB_CHOICE__

Usage:
  npm install -g yo generator-spring-bootstrap
  yo spring-bootstrap

Notes:
- The generator clones the template repository and replaces package and artifact placeholders.
- If git is available, the cloned .git will be removed and a fresh git repo will be initialized if requested.
